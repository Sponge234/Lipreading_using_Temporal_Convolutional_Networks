## 🚀 使用方法

### 快速启动
```bash
# 方式1: 使用启动脚本
./start_realtime.sh

# 方式2: 直接运行
python realtime_lipreading.py

# 方式3: 自定义参数
python realtime_lipreading.py \
    --camera-id 0 \
    --buffer-size 29 \
    --use-mediapipe
```

### 运行时控制
- **q**: 退出程序
- **r**: 重置帧缓冲
- **s**: 保存当前帧

## 🔍 工作流程

### 启动流程
```
1. 加载模型和配置
2. 初始化人脸检测器
3. 打开摄像头
4. 开始实时循环
```

### 检测循环
```
while True:
    1. 捕获视频帧
    2. 检测人脸和嘴部
    3. 提取嘴部ROI
    4. 添加到缓冲
    5. 缓冲满时推理
    6. 显示结果
    7. 处理键盘输入
```

### 推理流程
```
1. 预处理帧序列
2. 转换为tensor
3. GPU推理
4. 计算概率分布
5. 获取Top-5结果
```

## 🐛 故障排除

### 常见问题
1. **摄像头无法打开**: 检查设备权限和ID
2. **FPS过低**: 确认GPU可用，关闭其他程序
3. **检测不到人脸**: 改善光照，正脸朝向
4. **预测不准确**: 确保嘴部动作清晰

### 解决方案
详见 `REALTIME_GUIDE.md` 的故障排除章节
